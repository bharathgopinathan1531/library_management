# Library Management System

## Features
- Book CRUD Operations
- Member CRUD Operations
- Borrow Book
- Return Book
- Search Books
- Pagination
- Error Handling
- Swagger Documentation

## Tech Stack
- Python
- FastAPI
- SQLAlchemy
- SQLite

## Run

pip install -r requirements.txt

python -m uvicorn main:app --reload
